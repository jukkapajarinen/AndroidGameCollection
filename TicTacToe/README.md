# Tic Tac Toe

This is an android-sdk game made with event-based layouts.

## Play-store description

Save paper and pencils with this game!  
Try to line-up three X's or O's in a row to win.

Notice the following rules:  
1. Board size is 3 x 3.  
2. Line-up 3 X's or O's to win.

There's also a mysterious code. What could it be?

Enjoy the game!
